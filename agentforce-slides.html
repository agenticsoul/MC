02 / 04 — Maturity Model

Proposed Operating Model

Agent Factory

A dedicated internal capability to discover, build, govern, and scale agents systematically — rather than one-off pilots.

🔍
Discover & Prioritise
Run use-case discovery workshops with business units. Score candidates by effort vs. ROI. Maintain a live backlog of agent opportunities.
Centre of Excellence
🏗️
Build & Template
Reusable Agent Templates in Agent Builder. Standard Topics, Actions, and prompt libraries. Low-code first; Apex where needed.
Agentforce Builder
🛡️
Govern & Trust
Guardrails, escalation paths, and audit logs built-in. Einstein Trust Layer for data privacy. Human review gates for high-risk decisions.
Trust Layer
📊
Measure & Iterate
Agent performance dashboards in CRM Analytics. Track containment rate, CSAT, task completion, and time-to-resolution per agent.
CRM Analytics
🔗
Integrate & Extend
Connect to MuleSoft APIs, Data Cloud, Slack, and third-party systems. Agent Network for cross-cloud orchestration and delegation.
MuleSoft · Data Cloud
🚀
Scale & Operationalise
Promote agents from sandbox to production via DevOps Center. Maintain an agent catalogue. Drive adoption with enablement programmes.
DevOps Center
